Учет Весовых Операций

Описание:
Приложение для учета весовых операций транспорта с графическим интерфейсом на WPF. Позволяет отслеживать вес грузов, время отправки и прибытия транспортных средств.

Основные возможности:
1) Просмотр записей в табличном виде с фильтрацией по машинам
2) Добавление новых записей через диалоговое окно
3) Удаление записей с подтверждением
4) Импорт/экспорт данных в формате CSV
5) Логирование операций и ошибок
6) Асинхронная работа с базой данных
